# LINN — Legacy Intelligent Neural Network

Game prototype for IT Professional Day. Top-down shooter with an AI companion (LINN) that learns words and lets you combine them into abilities through a bullet-time prompt system.

**Play online:** https://realgamesbr.github.io/linn-game/

## Mechanics
- **Mouse / touch** to move and shoot
- **SPACE** or **◆** button to open the bullet-time prompt
- Combine words LINN has learned to cast unique abilities
- Survive 8 progressively harder waves

## Stack
Single-file HTML5 + Canvas + Web Audio API. No dependencies, no build step.

Built with Real Games Studio + Claude Code.
